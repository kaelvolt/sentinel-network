# Core Features of Sentinel Network

1. **Source Monitoring**: Continuously monitor various sources for relevant information.
2. **Signal Detection**: Identify and classify signals from the monitored sources.
3. **Claim Clustering**: Group similar claims to reduce noise and improve clarity.
4. **Alert Generation**: Generate alerts based on detected signals and clustered claims.
5. **User Interface**: Provide a user-friendly interface for users to interact with the platform.
