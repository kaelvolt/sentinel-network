# Sentinel Network

## Overview

Sentinel Network is an autonomous civic intelligence platform that monitors sources, detects signals, clusters claims, and generates alerts.

## Project Structure
- `src/`: Source code
- `assets/`: Assets like images and files
- `config/`: Configuration files

## Getting Started

To get started, run `npm install` to install dependencies.

## Usage

To run the application, execute:
```
npm start
```

## Features
- Source Monitoring
- Signal Detection
- Claim Clustering
- Alert Generation
- User Interface
